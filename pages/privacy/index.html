2. About this project

Videoflix is a private portfolio/demo project (not a commercial service), built as part of a fullstack developer training program. This notice explains what happens to your data when you use the demo.

3. Hosting & server logs

This site runs on my own server. Every request is logged with technical access data (IP address, timestamp, requested page, browser type) for security and operational purposes only. These logs are not combined with any other data source.

4. Account & authentication

Creating a demo account stores your email address and a hashed password. Login uses a JSON Web Token stored in an HttpOnly cookie, so the token is never exposed to client-side scripts. Account activation and password reset are handled by email links valid for a limited time. Legal basis: Art. 6(1)(b) GDPR (performance of the demo "contract" you request by signing up).

5. Uploaded videos

Videos uploaded for the demo are processed in the background (thumbnail generation, HLS transcoding) and stored on the same server. They are not shared with third parties.

6. Retention & deletion

Demo accounts and uploaded content are not kept indefinitely and can be deleted on request at any time by emailing the address above.

7. Your rights

You have the right to access, rectify, erase, and restrict processing of your personal data, and the right to lodge a complaint with a data protection supervisory authority. Contact the address above to exercise these rights.
